## ＜提案＞
　スキルバッジ
 保有する社内資格、スキルを一覧管理、見える化できるサービス

##  ＜提案の背景＞
　従業員間でお互いの保有資格、スキルを簡単に知る術が無い
　保有資格、スキルを紹介するワンストップの仕組みがない

## ＜ユースケース＞
　キャリア形成
　・上司面談時に新取得したバッジを紹介しながら、スキル向上をアピール
　・バッジ取得状況を説明しながら、挑戦したい分野の相談

## ＜MVP＞
・ユーザー：自身の保有バッジが確認できる
・バッジ発行：管理者アカウントのみ
・ユーザー：DIGメンバー
・画面構成：ログイン画面（認証機能）、バッジ一覧画面、バッジ付与画面
・技術要件：AWS利用、認証
